processor_architecture: u16

The system's processor architecture

Known values are defined in ProcessorArchitecture.

processor_level: u16

x86 (5 = 586, 6 = 686 ...) or ARM (6 = ARMv6, 7 = ARMv7 ...) CPU level

processor_revision: u16
